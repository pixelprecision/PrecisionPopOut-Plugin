Precision Pop Out Plugin - Documentation

This is the main page. Watch as it slides out of the way of the popout.

Instructions:

Include the latest version of jQuery in the head tag as well as the minified plugin.

                <script type="text/javascript" src="http://ajax.googleapis.com/ajax/libs/jquery/1.10.1/jquery.js"></script>
                <script type="text/javascript" src="jquery.precision-popout-1.2.min.js"></script>

Invoke the plugin on whatever you want to encapsulate in a pop out like so. Just put in the ID or Class of the html tag you are encapsulating. You don't have to include all of the options, but they are there to customize your pop out.

                <script>
                    $('#Demo2').precisionPopOut({
                            loadTweenMax: true, //=Loads the animation library
                            loadRotatePlugin: true, //=Loads the rotate text library
                            inlineCSS: true,
                            markUpInc: false, //=Set to true to use your own button
                            blackOut: true, //=Set to true to enable a fade out
                            speed: 0.35, //=Set your pop outs speed
                            responsive: true, //=Would you like the pop out to be responsive?
                            resWindowWidth: 900, //=What window width would you like the responsive pop out to show at?
                            popup: false, //=Would you like the pop out to be a pop up instead?
                            popClickText: 'Change Text to Whatever You Want', //=Set your button text here
                            popOutDirection: 'left', //=Set which side of screen you want pop out on ('top', 'right', 'left', 'bottom')
                            clickDefaultPos: 'top', //=Set which side of screen you want button on for desktop ('top', 'right', 'left', 'bottom')
                            clickResPos: 'bottom', //=Set which side of screen you want button on for mobile/responsive ('top', 'right', 'left', 'bottom')
                            width: '40%', //=Set the width of the pop out. Can be percentage or pixels (ie. '100%', or '500px')
                            height: '100%', //=Set the height of the pop out. Can be percentage or pixels (ie. '60%', or '300px')
                            backgroundColor: '#333', //=Set the color of the pop out. Can be hex or color (ie. '#ffd500', or 'black')
                            textColor: '#fff', //=Set the color of the button text. Can be hex or color (ie. '#333333', or 'white')
                            padding: 10, //=Set the padding of the pop out. Can be integer or css style (ie. 30, or '30px')
                            container: '#container', //=Set the outermost container you have that you want to slide out of the way. Can be ID, Class, or tag (ie. '#Wrapper', '.mainContent', or 'body')
                            clickBackground: '#333', //=Set the color of the button. Can be hex or color (ie. '#ffd500', or 'black')
                            clickColor: '#fff', //=Set the color of the button text. Can be hex or color (ie. '#ffd500', or 'black')
                            clickHeightV: 250, //=Set the height of the button if on side (vertical button). Can be integer or css style (ie. 300, or '300px')
                            clickWidthV: 30, //=Set the width of the button if on side (vertical button). Can be integer or css style (ie. 300, or '300px')
                            clickHeightH: 30, //=Set the height of the button if on top/bottom (horizontal button). Can be integer or css style (ie. 300, or '300px')
                            clickWidthH: 280, //=Set the width of the button if on top/bottom (horizontal button). Can be integer or css style (ie. 300, or '300px')
                            clickRndCorners: true, //=Set to true if you want rounded corners. Can be integer or css style (ie. 300, or '300px')
                            clickRadius: 15, //=Set the radius of the rounded corners. Can be integer or css style (ie. 10, or '10px')
                            closePos: 'topRight', //=Set the position of the close button ('topLeft', 'topMiddle', 'topLeft', 'btmLeft', 'btmMiddle', 'btmTop')
                            closeBackground: 'none', //=Set the color of the close button. Can be hex or color (ie. '#ffd500', or 'black')
                            closeColor: '#fff' //=Set the color of the close button text. Can be hex or color (ie. '#ffd500', or 'black')
                    });
                </script>
